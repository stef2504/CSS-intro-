Other useful html elements

Even though this is a course on css, it is important to have a good grasp of html -- beyond the basic paragraph, header and list markup. Using the right element and then writing styles for particular elements eliminates the need for adding classes to generic elements so often.

Blockquote

Blockquote is a container for a paragraph or paragraphs and can include a paragraph for the source of the quote and a "cite" element for the title of a work the quote comes from. By default the "cite" element is styled with italics - and is more meaningful than just an "em" tag.

Get to know the range of html elements available and use them appropriately!
